# Raquel Martínez Micó | Studio de Belleza 💄

Sitio web de Raquel Martínez Micó, especialista en colorimetría y alisados de lujo en Madrid. Proyecto diseñado con una estética minimalista y funcional.

##  Características
- **Carrusel Automático:** Exhibición de procedimientos principales.
- **Menú Inteligente:** Navegación con indicador animado por secciones.
- **Testimonios Infinitos:** Sección "Marquee" con reseñas reales de Google.
- **Diseño Responsivo:** Optimizado para móviles y tablets.

##  Tecnologías
- HTML5 / CSS3 (Variables y Grid)
- JavaScript Vanilla
- Google Fonts (Playfair Display & Montserrat)
